Frequently Asked Questions

1. How do I start learning here?
Tap Get Started, enter your name and the default password, then the Learn section unlocks with lesson buttons and vocabulary cards.
2. What is inside each lesson?
Each lesson shows a set of words with meaning, pronunciation, and quick info buttons. You can open a modal for examples and synonyms.
3. How do I pick a lesson level?
Use the lesson buttons (Lesson-1, Lesson-2, etc.). They fetch the level data from the API and load the words for that level.
4. Can I hear pronunciations?
The volume icon on each card is reserved for listening practice. If audio is available, it will be connected there in future updates.
5. Why do I sometimes see blank meanings?
Some API items may not include a meaning. When that happens, a placeholder message appears so you know the data is missing, not your connection.
